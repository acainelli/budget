Define Codable structs for JSON:
```swift
struct ExportData: Codable {
    let version: Int
    let exportDate: String
    let budgets: [BudgetExport]
    let expenses: [ExpenseExport]
}
struct BudgetExport: Codable {
    let monthYear: String
    let income: Double
}
struct ExpenseExport: Codable {
    let amount: Double
    let category: String
    let date: String
    let notes: String
    let monthYear: String
}
```

Export logic (computed property returning Data):
- Create ISO8601DateFormatter for dates
- Map allBudgets to BudgetExport array
- Map allExpenses to ExpenseExport array (category as rawValue, date as ISO8601 string)
- Create ExportData(version: 1, exportDate: ISO8601DateFormatter().string(from: Date()), budgets: ..., expenses: ...)
- Encode with JSONEncoder (outputFormatting: [.prettyPrinted, .sortedKeys])
- Return the Data

Export file name: "BudgetApp-Export-\(DateFormatter(yyyy-MM-dd).string(from: Date())).json"

Create a computed property for the export file as a transferable item. Use ShareLink with a preview. The simplest approach:
- Create a computed `exportDocument: Data` property
- Use ShareLink("Export Data", item: exportJSONFile, preview: SharePreview("BudgetApp Export", image: Image(systemName: "square.and.arrow.up")))
- For ShareLink with file: create a temporary file URL and use `ShareLink(item: url)` — OR use the simpler approach of `ShareLink(item: jsonString, subject: Text("BudgetApp Export"))` for text sharing
- Best approach for JSON file: write Data to temp file, return URL, use ShareLink(item: url). Use a helper function that writes to FileManager.default.temporaryDirectory

Import logic:
- .fileImporter(isPresented: $showImporter, allowedContentTypes: [.json]) { result in ... }
