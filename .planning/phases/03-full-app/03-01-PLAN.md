Create BulkDeleteSheet.swift:
- @Environment(\.modelContext) var modelContext, @Environment(\.dismiss) var dismiss
- Takes `allExpenses: [Expense]` parameter (unfiltered, since bulk delete operates on absolute time ranges)
- @State var scope: BulkDeleteScope = .day (enum with cases day, week, month)
- Computed property `expensesToDelete: [Expense]` filters allExpenses based on scope:
  - .day: expenses where Calendar.current.isDateInToday(expense.date)
  - .week: expenses where expense.date is within Calendar.current.dateInterval(of: .weekOfYear, for: Date())!
  - .month: expenses where expense.date is within Calendar.current.dateInterval(of: .month, for: Date())!
- @State var showConfirmation = false
- UI: NavigationStack with .navigationTitle("Bulk Delete")
  - Picker("Scope", selection: $scope) with .pickerStyle(.segmented) for Day | Week | Month
  - Text showing "\(expensesToDelete.count) expenses will be deleted" in .headline
  - Preview list of expenses (just category icon + amount, .font(.caption))
  - Red destructive Button "Delete \(expensesToDelete.count) Expenses" — sets showConfirmation = true
  - .confirmationDialog: on confirm, loop expensesToDelete calling modelContext.delete($0), then try? modelContext.save(), HapticManager.warning(), dismiss()
- After deletion, show a brief overlay toast. Simplest approach: set @State var deletedCount, and in the parent ExpensesView show an overlay Text that auto-hides via .task { try? await Task.sleep(for: .seconds(2.5)); showToast = false }

Replace ExpensesView.swift stub with full implementation:
- @Query(sort: \Expense.date, order: .reverse) var allExpenses: [Expense]
- @Environment(\.modelContext) var modelContext
- @State var searchText = ""
- @State var showBulkDelete = false
- @State var toastMessage: String? = nil
- Computed `filteredExpenses`: if searchText is empty, return allExpenses; otherwise filter where expense.notes.localizedCaseInsensitiveContains(searchText) || expense.category.displayName.localizedCaseInsensitiveContains(searchText)
- Computed `groupedByDay`: Dictionary(grouping: filteredExpenses) { Calendar.current.startOfDay(for: $0.date) } — then sorted by key descending
- NavigationStack containing List with ForEach over groupedByDay keys:
  - Section header: HStack with date label (formatted "EEEE, d MMMM" or similar medium style) on left, daily total (sum of section expenses).formattedEUR on right
  - ForEach expenses in section: NavigationLink(value: expense) wrapping ExpenseRowView(expense: expense)
  - .onDelete on inner ForEach: swipe-to-delete with modelContext.delete, try? modelContext.save(), HapticManager.warning()
- .navigationDestination(for: Expense.self) { expense in ExpenseDetailView(expense: expense) }
- .searchable(text: $searchText, prompt: "Search expenses")
- .toolbar: Button with trash icon (systemName "trash") that sets showBulkDelete = true
- .sheet(isPresented: $showBulkDelete) { BulkDeleteSheet(allExpenses: allExpenses) }
- .overlay for toast: if let toastMessage, show Text(toastMessage) in capsule at bottom, animate in/out
- .navigationTitle("Expenses")
