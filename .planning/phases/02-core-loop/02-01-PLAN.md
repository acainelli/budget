Create two new SwiftUI view files:

**NumpadView.swift:**
- Takes `@Binding var amountInCents: Int` — the raw integer cents value
- 4x3 LazyVGrid with columns [GridItem](repeating, count: 3, .flexible())
- Buttons: [1,2,3], [4,5,6], [7,8,9], [00, 0, backspace.fill (SF Symbol)]
- ATM-style (cents-first) logic:
  - Digit tap: `amountInCents = amountInCents * 10 + digit` (for "00": `* 100`)
  - Backspace: `amountInCents = amountInCents / 10`
  - Cap at 9_999_999 (7 digits = max €99.999,99) — ignore taps beyond cap
- Each button: large tappable area (minHeight 56), rounded rect background (.secondary.opacity(0.15)), cornerRadius DesignTokens.CornerRadius.small
- HapticManager.impact() on each digit tap, HapticManager.selection() on backspace
- Button text: .title2 weight .semibold for digits, SF Symbol for backspace

**AmountDisplayView.swift:**
- Takes `amountInCents: Int`
- Computes `Double(amountInCents) / 100.0` and formats via `.formattedEUR`
- Large centered text: .system(size: 48, weight: .semibold, design: .rounded).monospacedDigit()
- When amountInCents == 0, show "€0,00" in .secondary color; otherwise .primary
